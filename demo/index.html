Icoonies

This is a demo. For more information and examples on icoonies see the project on github:

Important: If you don't see the svg images, and this demo running from your local computer without the server, open this page with the browser Mozilla Firefox. If you don't see the svg images, and this demo running from the server, check that the configuration of required file extension .svg is enabled.

Example

The following examples are some solutions that can offer icoonies. Your imagination can do much more.

  • Mode background svg
  • Mode image svg
  • Mode svg

Mode background svg

In these examples the svg is managed by icoonies as background element.

Show areas of the image

Icoonies identifies areas of svg. For each area is assigned an id with prefix "icoo-area-(n)".

Click on the images of the example to see the areas that made up.
